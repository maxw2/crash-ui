<swiper />

## 代码演示

``` html
<div class="swiper">
    <ul class="content">
      <li>1</li>
      <li>2</li>
      <li>3</li>
    </ul>
</div>
```

``` js
import { Swiper } from 'crash-ui'

let swiper = new Swiper('.swiper')
```

## API

| 方法 |  描述  | 属性 |
|------|:------:|:----:|
| next | 下一页 | None |
| prev | 上一页 | None |
